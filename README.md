<h1 align = "center"> Projeto Cientista de Dados</h1>
Usando o conjunto de dados públicos de comércio eletrônico brasileiro da Olist. 
O conjunto possui mais de 100mil pedios feitos entre os anos de 2016 e 2018. 
Estes são dados comerciais reais, que foram anonimizados, e as referências às empresas e parceiros no texto da análise foram substituídas pelos nomes das grandes casas de Game of Thrones.
